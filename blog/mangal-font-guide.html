Mangal Font: Complete Guide

You've probably typed in Mangal without knowing it. Here's what this font actually is, why it's on almost every Windows PC, and when to use something else.

7 min read·July 19, 2026

If you've ever opened MS Word on a Windows computer, typed in Hindi, and it just worked without installing anything — that was Mangal. It's one of those fonts almost everyone in India has used at some point without ever knowing its name, because it's been bundled into Windows since Windows XP.

What Is Mangal?

Mangal is a Unicode Devanagari font developed by Microsoft, included as a default system font on Windows. Unlike Kruti Dev, which maps Hindi glyphs onto English keyboard positions using a custom encoding, Mangal uses proper Unicode — meaning the text you type is stored as actual Devanagari characters, not disguised English letters.

This one difference is what makes Mangal usable everywhere: on websites, in search engines, across different operating systems, and on phones — without needing the exact same font file installed on every device.

Why Mangal Is So Common

  • It ships by default with Windows, so almost every Indian PC already has it
  • It was one of the first properly Unicode-compliant Hindi fonts widely available for free
  • Government forms, official documents, and many school/college assignments default to it
  • It works out of the box in MS Word, Excel, and PowerPoint without any setup

Mangal vs Kruti Dev: The Real Difference

Feature Mangal Kruti Dev
Encoding Unicode Custom (ASCII-based)
Works on the web Yes, natively No, needs conversion
Searchable text Yes No
Visual style Plain, functional Multiple bold/decorative variants
Best for Official docs, forms, web content Legacy print, older DTP workflows

Neither font is objectively "better" — they're built for different jobs. If you're filling a government form or writing something that needs to be searchable and shareable digitally, Mangal (or another Unicode font) is the right call. If you're working with an old printing setup that's already built around Kruti Dev, switching isn't always practical.

Where to Get Mangal

Since Mangal ships with Windows, most people already have it without downloading anything. To check:

  1. Open MS Word
  2. Click the font dropdown
  3. Type "Mangal" — if it's installed, it will appear immediately

If it's genuinely missing (rare, but happens on some stripped-down Windows installs or on Mac/Linux), Microsoft's official Devanagari fonts are also downloadable through Windows Update or the Microsoft Store's language pack for Hindi.

Good to know: If you're on Mac or Linux and don't have Mangal, Noto Sans Devanagari is a free, near-identical looking alternative that covers the same use cases and is available from Google Fonts on any operating system.

Limitations of Mangal

Mangal is reliable, but it's not particularly attractive by modern design standards. It was built for function — official documents and forms — not for headlines, posters, or branding. If you're designing something that needs personality (a poster, an Instagram post, a wedding card), Mangal alone will look flat. Pair it with a decorative font for headlines, and keep Mangal for the plain, readable body text.

Quick Summary

  • Mangal is a free, Unicode Devanagari font built into Windows since Windows XP
  • It's the safest choice for anything official, searchable, or web-based
  • It's not decorative — use it for body text, not headlines
  • Kruti Dev still exists for older print workflows, but isn't Unicode and needs conversion before digital use
